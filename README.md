# Zepto Inventory Analysis using SQL

## Project Overview

This project analyzes Zepto inventory data using SQL to understand product pricing, stock availability, discounts, and category-level business insights.

## Dataset Columns

* category
* name
* mrp
* discountPercent
* availableQuantity
* discountedSellingPrice
* weightInGms
* outOfStock
* quantity

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* HAVING
* CASE WHEN
* UPDATE
* DELETE
* Aggregate Functions
* Window Functions
* DENSE_RANK()

## Business Questions Solved

1. Products in stock vs out of stock
2. Duplicate product names
3. Top discount products
4. High MRP out-of-stock products
5. Category revenue estimation
6. Average discount by category
7. Price per gram analysis
8. Weight classification
9. Ranking discount products
10. Highest priced product in each category

## Key Insights

* Identified categories with highest discounts
* Found premium products with low discounts
* Estimated inventory revenue by category
* Ranked products using window functions
